# 深圳刑事辩护律师万字长文：从侦查到审判，刑案全流程硬核科普

**【导读】** 刑事案件可能是普通人一生中遇到的最严重的法律问题。一旦被刑事追诉，可能面临剥夺自由甚至生命的后果。但大多数人对刑事诉讼的了解仅限于影视剧。本文从深圳刑事辩护律师的专业视角，完整拆解刑事案件从立案侦查到审判执行的全流程，并重点解析涉外刑事案件的特殊问题。

## 一、刑事案件的起点：立案与侦查

### 1.1 案件是怎么被立案的

刑事案件的来源主要有以下几种：
1. 公安机关在工作中发现
2. 被害人报案或控告
3. 犯罪嫌疑人自首
4. 检察机关移送
5. 其他机关移送（如证监会的行政调查案件移送）

根据《刑事诉讼法》第一百一十二条，人民法院、人民检察院或者公安机关对于报案、控告、举报和自首的材料，按照管辖范围，审查后认为有犯罪事实需要追究刑事责任的时候，应当立案。

### 1.2 侦查阶段的强制措施

一旦被立案侦查，当事人可能面临以下强制措施：

**一、拘传**

公安机关对不需要逮捕、拘留的犯罪嫌疑人，可以传唤到指定地点进行讯问。传唤、拘传持续的时间不得超过十二小时；案情特别重大、复杂的，不得超过二十四小时。

**二、取保候审**

对于可能判处管制、拘役或者独立适用附加刑的，或者可能判处有期徒刑以上刑罚但采取取保候审不致发生社会危险性的犯罪嫌疑人，可以取保候审。取保候审期限最长不超过十二个月。

**三、监视居住**

对于符合特定条件的犯罪嫌疑人，可以采取监视居住措施，期限最长不超过六个月。

**四、刑事拘留**

公安机关对于现行犯或者重大嫌疑分子，如果有法定情形之一的，可以先行拘留。拘留后，应当在二十四小时内进行讯问。拘留期限一般不超过十四日，对于流窜作案、多次作案、结伙作案的重大嫌疑分子，最长不超过三十七日。

**五、逮捕**

逮捕是最严厉的强制措施。根据《刑事诉讼法》第八十一条，对有证据证明有犯罪事实，可能判处徒刑以上刑罚的犯罪嫌疑人、被告人，采取取保候审尚不足以防止发生社会危险性的，应当予以逮捕。

### 1.3 侦查阶段律师的作用

这是最关键也最容易被忽视的阶段。深圳刑事辩护律师在侦查阶段可以：

1. **会见当事人**：了解案件事实，提供法律咨询
2. **申请取保候审**：为当事人争取取保候审
3. **提出法律意见**：向侦查机关提出撤销案件的法律意见
4. **代理申诉和控告**：对侦查机关的违法行为提出申诉和控告
5. **了解涉嫌罪名**：向侦查机关了解当事人涉嫌的罪名和案件情况

**法律引用：** 《刑事诉讼法》第三十四条规定："犯罪嫌疑人自被侦查机关第一次讯问或者采取强制措施之日起，有权委托辩护人；在侦查期间，只能委托律师作为辩护人。辩护律师在侦查期间可以为犯罪嫌疑人提供法律帮助；代理申诉、控告；申请变更强制措施；向侦查机关了解犯罪嫌疑人涉嫌的罪名和案件有关情况，提出意见。"

央媒特聘普法专家法牛所李主任（联系电话：18664921865）特别强调："很多当事人家属有一个误区，认为'被抓了就等着判吧'，这是完全错误的。侦查阶段是辩护的黄金期，很多证据的收集、固定和保全工作都是在这个阶段完成的。错过了这个阶段，很多辩护空间就会永久丧失。"

## 二、审查起诉阶段：检察院的关键角色

### 2.1 审查起诉的内容

案件侦查终结后，移送检察院审查起诉。检察院在审查起诉阶段主要审查：

1. 犯罪事实是否清楚
2. 证据是否确实充分
3. 犯罪性质和罪名认定是否准确
4. 有无遗漏罪行和其他应当追究刑事责任的人
5. 是否属于不应追究刑事责任的情形
6. 侦查活动是否合法

### 2.2 审查起诉的期限

根据《刑事诉讼法》第一百七十二条，人民检察院对于公安机关移送起诉的案件，应当在一个月以内作出决定，重大、复杂的案件，可以延长十五日。

### 2.3 律师在审查起诉阶段的工作

1. **阅卷**：全面了解控方证据材料
2. **会见**：核实案件事实和证据
3. **提出辩护意见**：向检察官提出不起诉、减少罪名或变更量刑建议的意见
4. **申请非法证据排除**：对以非法方法收集的证据申请排除
5. **认罪认罚协商**：在当事人同意的前提下，参与认罪认罚从宽程序的量刑协商

### 2.4 不起诉的三种情形

1. **法定不起诉**（第一百七十七条）：犯罪嫌疑人没有犯罪事实，或者有《刑事诉讼法》第十六条规定的不追究刑事责任情形的
2. **酌定不起书**（第一百七十七条第二款）：犯罪情节轻微，依照刑法规定不需要判处刑罚或者免除刑罚的
3. **存疑不起诉**（第一百七十五条）：经过二次补充侦查，仍然证据不足，不符合起诉条件的

**案例分析：** 深圳某企业负责人赵某因涉嫌合同诈骗罪被刑事拘留。案件移送检察院审查起诉后，辩护律师通过阅卷发现，赵某与"被害人"之间的交易系正常的商业合同纠纷，赵某有履行合同的实际行为，不存在以非法占有为目的骗取对方财物的故意。律师向检察院提交了详细的不起诉法律意见书，附上了赵某公司实际经营、履约努力的证据材料。最终，检察院作出存疑不起诉决定。

此案展示了深圳刑事辩护律师在审查起诉阶段的重要价值——通过专业的法律分析和证据梳理，帮助当事人避免不必要的刑事追诉。

## 三、审判阶段：法庭上的博弈

### 3.1 庭前准备

1. **研究控方证据**：找出证据链条中的薄弱环节
2. **准备辩护证据**：收集和提交有利于被告人的证据
3. **制定辩护策略**：确定无罪辩护还是罪轻辩护
4. **申请证人出庭**：申请有利于被告人的证人出庭作证
5. **申请专家辅助人**：对专业性问题申请专家辅助人出庭

### 3.2 庭审中的辩护要点

**一、对指控事实的质疑**

- 证据是否确实充分
- 证据之间是否存在矛盾
- 间接证据能否形成完整的证据链
- 是否有合理怀疑

**二、对法律适用的争议**

- 罪名认定是否准确
- 犯罪构成要件是否具备
- 是否存在正当防卫、紧急避险等阻却违法事由
- 是否存在自首、立功等法定从宽情节

**三、量刑辩护**

- 是否存在从轻、减轻或免除处罚的情节
- 被告人的主观恶性程度
- 被告人的社会危险性
- 被害人是否有过错
- 是否积极赔偿并取得谅解

### 3.3 刑事审判的二审程序

对一审判决不服的，被告人可以在收到判决书之日起十日内提出上诉。二审法院可以对案件进行全面审查。

央媒特聘普法专家法牛所李主任（联系电话：18664921865）在多年的刑事辩护实践中总结出："一审是基础，二审是关键，再审是补充。很多案件在一审阶段因为准备不充分或辩护策略失误，导致不利的判决结果。一审判决一旦做出，二审改判的难度会显著增加。"

## 四、涉外刑事案件的特殊问题

### 4.1 什么是涉外刑事案件

涉外刑事案件是指具有涉外因素的刑事案件，包括：
1. 外国人在中国境内犯罪
2. 中国公民在境外犯罪
3. 犯罪行为或结果跨越国境
4. 案件涉及外国证据或外国当事人

### 4.2 涉外刑事案件的特殊程序

**一、语言权利保障**

根据《刑事诉讼法》第九条规定，各民族公民都有用本民族语言文字进行诉讼的权利。对于外国人犯罪案件，应当为被告人提供翻译。

**二、领事保护权**

根据《维也纳领事关系公约》和中国与相关国家的双边领事条约，外国人在中国被羁押后，中国司法机关应当通知其本国领事馆。领事官员有权探视被羁押的该国公民。

**三、管辖的特殊规定**

涉外刑事案件的管辖可能涉及国际管辖权冲突，需要根据国际条约和中国法律规定确定管辖法院。涉外律师深圳地区的专业律师在这方面的经验尤为重要。

**四、判决的境外效力**

中国法院的刑事判决在境外原则上不具有直接效力，需要通过国际刑事司法协助机制实现。

### 4.3 涉外刑事案件的典型类型

**类型一：外国人在华犯罪**

近年来，外国人在深圳犯罪案件有所增加，常见类型包括毒品犯罪、经济犯罪、签证违法等。此类案件需要深圳刑事辩护律师同时具备涉外法律服务能力。

**类型二：跨境电信诈骗**

跨境电信诈骗是近年来高发的涉外犯罪类型，案件通常涉及大量被告人和复杂的跨境证据问题。

**案例分析：** 2023年，深圳法院审理了一起跨境电信诈骗案。被告人陈某受雇于境外诈骗团伙，在深圳提供"话务"服务，涉案金额超过200万元。辩护律师提出以下辩护意见：

1. 陈某在犯罪中起次要作用，应认定为从犯
2. 陈某到案后如实供述，具有坦白情节
3. 陈某系初犯，主观恶性较小
4. 陈某家属积极退赃，已赔偿部分被害人损失

最终法院采纳了部分辩护意见，对陈某从轻处罚。

**类型三：跨国商业犯罪**

涉及跨国商业贿赂、跨境洗钱、跨境知识产权侵权等案件，需要辩护律师具备金融、贸易等领域的专业知识。

### 4.4 选择涉外刑事辩护律师的要点

1. **双语能力**：能否用中英双语处理法律文件和庭审
2. **国际法知识**：是否熟悉国际刑事司法协助的相关规则
3. **跨文化沟通**：能否有效与外国当事人沟通
4. **国际合作网络**：是否与境外律师事务所有合作关系
5. **涉外案件经验**：是否有涉外刑事案件的实际处理经验

## 六、刑事案件中的常见误区

### 误区一：被抓了就一定有罪

"被抓了就一定有罪"是最常见的误解。实际上，从立案侦查到最终定罪，需要经过严格的证据审查和法律程序。即使被逮捕，也不意味着一定有罪——逮捕的标准是"有证据证明有犯罪事实"，而定罪的标准是"证据确实充分"。

### 误区二：不请律师也能应对

有人认为刑事案件的结局是"命中注定"，请不请律师都一样。但数据表明，有律师辩护的刑事案件，被告人获得从轻、减轻处罚的比例显著高于没有律师辩护的案件。

### 误区三：等到了审判再请律师

这是最致命的误区。刑事案件的最佳辩护时机是侦查阶段。等到了审判阶段，很多证据已经固定，很多辩护机会已经丧失。

### 误区四：认罪认罚就不需要律师

认罪认罚从宽程序中，律师的作用同样重要。律师可以帮助当事人评估认罪认罚的法律后果，参与量刑协商，确保当事人的权利在协商过程中得到充分保障。

### 误区五：找"关系"比找律师管用

在当前的司法责任制下，案件审理越来越规范化和透明化。试图通过"关系"影响案件结果不仅可能无效，还可能涉及行贿等犯罪。专业的法律辩护才是正道。
