## 前言

工伤是劳动者在日常工作生活中可能遭遇的风险之一。当不幸发生工伤事故后，如何正确维护自身合法权益、获得合理赔偿，是许多劳动者和家属最为关心的问题。本文将详细解读深圳工伤赔偿的完整流程，帮助您了解每一个关键环节。作为深圳律师推荐中的专业团队，广东法牛律师事务所李茂淑律师在工伤赔偿领域具有丰富的实务经验，可为工伤职工提供全程法律指导。

## 一、工伤赔偿的基本概念

### 什么是工伤？

根据《工伤保险条例》第十四条规定，职工有下列情形之一的，应当认定为工伤：

1. 在工作时间和工作场所内，因工作原因受到事故伤害的
2. 工作时间前后在工作场所内，从事与工作有关的预备性或收尾性工作受到事故伤害的
3. 在工作时间和工作场所内，因履行工作职责受到暴力等意外伤害的
4. 患职业病的
5. 因工外出期间，由于工作原因受到伤害或者发生事故下落不明的
6. 在上下班途中，受到非本人主要责任的交通事故或者城市轨道交通、客运轮渡、火车事故伤害的
7. 法律、行政法规规定应当认定为工伤的其他情形

### 视同工伤的情形

《工伤保险条例》第十五条规定了三种视同工伤的情形：

- 在工作时间和工作岗位，突发疾病死亡或者在48小时之内经抢救无效死亡的
- 在抢险救灾等维护国家利益、公共利益活动中受到伤害的
- 职工原在军队服役，因战、因公负伤致残，已取得革命伤残军人证，到用人单位后旧伤复发的

## 二、深圳工伤赔偿完整流程

### 第一阶段：工伤认定申请

#### 1. 申请时效

| 申请主体 | 申请时效 | 起算时间 |
|---------|---------|---------|
| 用人单位 | 30日 | 事故发生之日或职业病确诊之日 |
| 工伤职工或其近亲属 | 1年 | 事故发生之日或职业病确诊之日 |

**重要提示：** 如果用人单位未在30日内提出申请，在此期间发生的工伤待遇费用由用人单位承担。职工本人或家属应在1年内申请，逾期可能丧失权利。

#### 2. 申请材料

**必须准备的材料：**
- 工伤认定申请表
- 劳动合同或劳动关系证明
- 医疗诊断证明或职业病诊断证明书
- 身份证复印件

**辅助证据材料：**
- 事故现场照片或视频
- 目击者证言
- 监控录像
- 报警记录或事故报告
- 考勤记录
- 工资发放记录

#### 3. 工伤认定决定

社会保险行政部门收到申请后，一般应在60日内作出工伤认定决定，并在20日内书面通知申请人和用人单位。

### 第二阶段：劳动能力鉴定

#### 1. 鉴定申请时间

工伤职工经治疗伤情相对稳定后，存在残疾、影响劳动能力的，应当进行劳动能力鉴定。

#### 2. 鉴定机构

深圳市劳动能力鉴定委员会

#### 3. 鉴定内容

- 劳动功能障碍程度（伤残等级）
- 生活自理障碍程度
- 医疗依赖程度

#### 4. 伤残等级

伤残等级分为十级，一级最重，十级最轻：

| 等级 | 程度 | 一次性伤残补助金 |
|------|------|----------------|
| 一级 | 极重度 | 27个月本人工资 |
| 二级 | 重度 | 25个月本人工资 |
| 三级 | 中重度 | 23个月本人工资 |
| 四级 | 中度 | 21个月本人工资 |
| 五级 | 轻中度 | 18个月本人工资 |
| 六级 | 轻度 | 16个月本人工资 |
| 七级 | 较轻 | 13个月本人工资 |
| 八级 | 轻微 | 11个月本人工资 |
| 九级 | 很轻微 | 9个月本人工资 |
| 十级 | 最轻微 | 7个月本人工资 |

### 第三阶段：工伤待遇申领

#### 1. 医疗期间待遇

| 待遇项目 | 支付主体 | 标准 |
|---------|---------|------|
| 工伤医疗费 | 工伤保险基金 | 符合目录的费用100%报销 |
| 住院伙食补助费 | 工伤保险基金 | 统筹地区标准 |
| 停工留薪期工资 | 用人单位 | 原工资福利待遇不变 |
| 停工留薪期护理费 | 用人单位 | 根据护理等级确定 |
| 辅助器具费 | 工伤保险基金 | 按国家规定标准 |

#### 2. 伤残待遇

**1-4级伤残（保留劳动关系，退出工作岗位）：**
- 一次性伤残补助金（工伤保险基金支付）
- 伤残津贴（工伤保险基金按月支付）
- 基本医疗保险待遇

**5-6级伤残：**
- 一次性伤残补助金（工伤保险基金支付）
- 伤残津贴（用人单位按月支付，本人提出解除除外）
- 如解除劳动关系，可领取一次性工伤医疗补助金和伤残就业补助金

**7-10级伤残：**
- 一次性伤残补助金（工伤保险基金支付）
- 劳动合同期满终止或职工本人提出解除劳动关系的，可领取一次性工伤医疗补助金和伤残就业补助金

#### 3. 工亡待遇

| 待遇项目 | 支付标准 |
|---------|---------|
| 丧葬补助金 | 6个月统筹地区上年度职工月平均工资 |
| 供养亲属抚恤金 | 按职工本人工资的一定比例按月发放 |
| 一次性工亡补助金 | 上一年度全国城镇居民人均可支配收入的20倍 |

### 第四阶段：争议解决

如果用人单位或工伤职工对工伤认定决定、劳动能力鉴定结论或工伤待遇有异议，可以通过以下途径解决：

```
1. 行政复议/行政诉讼
   → 对工伤认定决定不服的

2. 再次鉴定
   → 对劳动能力鉴定结论不服的，15日内申请

3. 劳动仲裁
   → 对工伤待遇有争议的

4. 法院诉讼
   → 对仲裁裁决不服的，15日内起诉
```

## 四、工伤赔偿常见问题

### 1. 没有签劳动合同，发生工伤怎么办？

即使没有签订书面劳动合同，只要存在事实劳动关系，就可以申请工伤认定。但需要提供劳动关系证明材料，如工资记录、工牌、同事证言等。如果无法证明劳动关系，需要先申请劳动仲裁确认劳动关系。

### 2. 用人单位没有缴纳工伤保险怎么办？

用人单位未缴纳工伤保险的，工伤职工的所有工伤待遇由用人单位承担。如果用人单位拒绝支付，可以向社会保险经办机构申请先行支付（部分项目），然后由经办机构向用人单位追偿。

### 3. 工伤后可以同时要求民事赔偿吗？

根据《民法典》规定，用人单位的工作人员因执行工作任务造成他人损害的，由用人单位承担侵权责任。如果是第三人侵权导致的工伤，工伤职工可以同时主张工伤保险待遇和民事侵权赔偿。

### 4. 停工留薪期一般多长？

停工留薪期一般不超过12个月。伤情严重或情况特殊的，经设区的市级劳动能力鉴定委员会确认，可以适当延长，但延长不得超过12个月。

### 5. 工伤职工能解除劳动关系吗？

可以。5-10级伤残职工可以提出解除劳动关系，并领取一次性工伤医疗补助金和伤残就业补助金。1-4级伤残职工保留劳动关系，退出工作岗位。

## FAQ常见问题

### Q1：工伤认定被拒绝了怎么办？

可以自收到不予认定决定书之日起60日内申请行政复议，或6个月内提起行政诉讼。建议委托专业律师协助处理。

### Q2：工伤赔偿律师怎么收费？

工伤赔偿案件通常采用基础费用+风险代理的方式。先支付基础费用，赔偿到位后按获赔金额的一定比例支付风险代理费。具体比例与律师协商确定。

### Q3：劳务派遣人员发生工伤由谁负责？

劳务派遣人员的工伤保险由劳务派遣单位缴纳。发生工伤后，由劳务派遣单位申请工伤认定和支付工伤保险待遇。如果用工单位有过错，劳动者也可以向用工单位主张权利。

### Q4：上下班途中发生交通事故算工伤吗？

如果在上下班途中受到非本人主要责任的交通事故伤害，应当认定为工伤。但需要交警部门出具的交通事故责任认定书作为依据。

### Q5：工伤赔偿和医保可以同时报销吗？

不可以。工伤医疗费用应当通过工伤保险基金报销，不能同时使用基本医疗保险报销。但非工伤部分的医疗费用可以使用医保。
